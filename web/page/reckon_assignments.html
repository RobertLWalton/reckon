RECKON Assignments

Single Variable Expression Assignments

Examples: 

X = 25
25

next X = X + 10
35

X
35

A simple assignment consists of a variable name (e.g., X) followed by the assignment operator (i.e., =) followed by an expression (e.g., 25 or X + 1). Storage for the variable named is allocated, the expression is evaluated, and the expression value is store in the variable storage. The name of the newly allocated variable does not become visible until after the statement (i.e., in X + 10 above, X refers to the variable allocated by X = 25).

The value of a variable cannot be changed. But a new variable of the same name can be created, and this will hide the old variable. To do this, the variable name must be preceded by next in the assignment for the new variable (e.g., next X).

Examples: 

X = 5
5

Z = 8
8

Y = Z
8

Multi-Variable Expression Assignments

Examples: 

X, Y = 5, 10
5
10

next X, next Y = X + Y, X * Y
15
50

A multi-variable expression assignment statement consists of a comma-separated list of variable names (e.g., X, Y) followed by the assignment operator (i.e., =) followed by a comma-separated list of expressions. The number of variables must equal the number of expressions. Storage is allocated for the variables, then the expressions are evaulated, and then the expression values are stored in the variable storage. The value of the first expression is stored in the first variable, the value of the second expression is stored in the second variable, and so forth. The names of the newly allocated variables do not become visible until after the statement (i.e., in X + Y and X * Y above, X and Y refer to the variables allocated by X, Y = 5, 10).

Just as in the single variable case, the values of variables cannot be changed, but new variables of the same name can be created and hide the old variables using next.

Examples: 

M, N = 5, 10
    // The values of M and N are printed in order
    // after the statement
5
10

next M, next N = N, M
10
5

P, Q = M * N, M + N
50
15

R, next P = P / Q, P + 1
3.33333333333333
51
