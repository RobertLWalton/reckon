RECKON Assignments

Simple Expression Assignments

Examples: 

X = 25
25

next X = X + 10
35

X
35

A simple assignment consists of a variable name (e.g., X) followed by the assignment operator (i.e., =) followed by an expression (e.g., 25 or X + 1). Storage for the variable named is allocated, the expression is evaluated, and the expression value is store in the variable storage.

The value of a variable cannot be changed. But a new variable of the same name can be created, and this will hide the old variable. To do this, the variable name must be preceded by next in the assignment for the new variable (e.g., next X).

Examples: 

X = 5
5

Z = 8
8

Y = Z
